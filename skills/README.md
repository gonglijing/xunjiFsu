# Project Skills

This folder contains reusable Codex skills for `fsu` engineering workflows.

## Skills

- `gogw-feature-refactor`: implement/refactor backend + frontend + wasm changes and run validation.
- `gogw-submodule-publisher`: publish `plugin_north`/`drvs` and update parent submodule pointers safely.
- `fsu-memory-refactor`: reduce allocations and steady-state memory in `fsu` hot paths.
